# Standardize old format ethnicity enrollment data

Converts 2021-2023 format (ID, District / School, Ethnicity) to 2024+
format (Type, id, District/School, Ethnicity).

## Usage

``` r
standardize_old_format_ethnicity(df)
```

## Arguments

- df:

  Data frame from old format file

## Value

Data frame with standardized column names
