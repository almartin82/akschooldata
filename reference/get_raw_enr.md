# Download raw enrollment data from Alaska DEED

Downloads school enrollment data directly from Alaska Department of
Education & Early Development (DEED). This function downloads both
grade-level and ethnicity enrollment files from DEED's statistics
portal.

## Usage

``` r
get_raw_enr(end_year)
```

## Arguments

- end_year:

  School year end (2023-24 = 2024)

## Value

List with school and district data frames
