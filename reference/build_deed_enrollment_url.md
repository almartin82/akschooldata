# Build Alaska DEED enrollment URL

Constructs URL for Alaska DEED enrollment data files.

## Usage

``` r
build_deed_enrollment_url(end_year, file_type = "grade")
```

## Arguments

- end_year:

  School year end (e.g., 2024 for 2023-24)

- file_type:

  Type of file: "grade" or "ethnicity"

## Value

URL string
